Catálogo de Autos - README

Este proyecto es un catálogo de autos que te permite agregar y mostrar diferentes tipos de autos, como sedanes, deportivos (sports) y SUVs. Cada auto tiene una serie de detalles, como nombre, marca, modelo, tipo, combustible, año, motor, potencia y precio.

Requisitos previos

Asegúrate de tener instalado Java Development Kit (JDK) en tu sistema. Si no lo tienes, puedes descargarlo desde el sitio web oficial de Oracle.

Cómo ejecutar el programa

---Descarga o clona este repositorio en tu computadora.
---Abre tu entorno de desarrollo de Java (por ejemplo, Eclipse, IntelliJ, NetBeans) y carga el proyecto.
---Encuentra el archivo CatalogoAutosPanel.java y ejecútalo como una aplicación Java.

Cómo utilizar el Catálogo de Autos

En la interfaz de usuario, selecciona el tipo de auto que deseas agregar (Sedan, Sport o SUV) en el cuadro desplegable "Elija un tipo de Auto".
Luego, ingresa los detalles del auto en los campos de texto provistos, como nombre, marca, modelo, tipo, combustible, año, motor, potencia y precio.
Haz clic en el botón "Agregar Auto" para agregar el auto al catálogo.
Una vez agregado el auto, podrás seleccionarlo en el cuadro desplegable "Elija un Auto del Catálogo".
Haz clic en el botón "Mostrar Auto" para ver los detalles del auto seleccionado en el panel de detalles.
Si deseas salir del programa, simplemente haz clic en el botón "Salir".
Notas adicionales
Si un auto tiene una imagen asociada, se mostrará en el panel principal. Para agregar una imagen a un auto, haz clic en el botón "Agregar Auto" y selecciona una imagen desde tu sistema de archivos.
Los autos se clasifican en tres tipos principales: sedanes, deportivos (sports) y SUVs. Puedes ver los autos de cada tipo seleccionando la opción correspondiente en el cuadro desplegable "Elija un tipo de Auto".
